# 📈 DecodeLabs Data Analytics Internship Portfolio

Hi, I'm Aleeza — this repository showcases the data analytics projects I completed during the DecodeLabs internship program, covering the full analytics workflow: data auditing, exploratory analysis, SQL querying, and business-focused reporting.

Each project tackles a different stage of the data analytics pipeline using a consistent e-commerce dataset, simulating how a real analytics team would clean, query, analyze, and report on operational data.

## 🗂️ Projects

| # | Project | Focus | Tools |
|---|---------|-------|-------|
| 1 | [Data Audit](./Project-1-Data-Audit) | Data quality auditing & cleaning with formal change-log tracking | Python, Pandas |
| 2 | [E-Commerce EDA — Outlier Detection](./Project-2-Ecommerce-EDA) | Statistical outlier detection (IQR vs. Z-Score) and revenue driver analysis | Python, Pandas, SciPy, Seaborn |
| 3 | [SQL Analysis](./Project-3-SQL) | Schema design and SQL querying for business reporting | SQL |
| 4 | [E-Commerce Sales Performance](./Project-4-Ecommerce-Sales) | Sales, marketing channel, and fulfillment performance analysis | Python, Pandas, Matplotlib |

## 🧠 Skills Demonstrated
- **Data Cleaning & Quality Auditing** — handling missing values, duplicates, and formatting inconsistencies with documented change tracking
- **Statistical Analysis** — outlier detection methods (IQR, Z-Score), correlation analysis
- **SQL** — schema design, aggregation, filtering, business querying
- **Exploratory Data Analysis** — distribution analysis, visualization, trend identification
- **Data Storytelling** — translating analysis into clear, actionable business recommendations

## 🔑 Highlighted Findings Across Projects
- Identified a **legitimate high-value "VIP" customer segment** through outlier analysis rather than dismissing it as bad data (Project 2).
- Flagged a **critical fulfillment issue**: cancelled and returned orders outnumbered successful deliveries in the sales dataset (Project 4).
- Found that **revenue is driven by order volume, not item price** — a key insight for pricing and discount strategy (Project 2).
- Pinpointed **Instagram and Email** as the top-performing customer acquisition channels (Project 4).
